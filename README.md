# 📈 Investra

Investra is a full-stack virtual stock trading platform built using the **MERN** stack, inspired by **Zerodha's user interface and user experience**.

The project consists of three independently deployed applications:

- **Landing Page** – Product information and user onboarding
- **Trading Dashboard** – Virtual trading, portfolio management, and analytics
- **Backend API** – Authentication, trading logic, and database operations

## 🚀 Live Demo

- **Landing Page:** https://investra-zerodha-clone1-hx2d.vercel.app
- **Trading Dashboard:** https://investra-zerodha-clone1.vercel.app

> **Note:** The backend is hosted on Render's free tier and may take 30–60 seconds to respond after inactivity.

---

## ✨ Features

- User Authentication & Authorization (JWT)
- Password Hashing with bcrypt
- HTTP-only Cookie Authentication
- Virtual Wallet
- Buy & Sell Stocks
- Holdings & Portfolio Management
- Order History
- Portfolio Analytics
- Interactive Charts
- Stock Market News Integration
